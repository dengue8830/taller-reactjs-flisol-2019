Basado en el estado
```notes
Tu componente reacciona a cambios de estado
```

---

<LiveCode title="Reacciona a cambios de estado" code={`
class Ejemplo extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      nombre: '-'
    }
  }
  render() {
    return (
      <div>
        <input
          onChange={e => this.setState({ nombre: e.currentTarget.value })}
          placeholder='Escribe algo' 
        />
        <label 
          style={{ color: '#212121' }}>
           hola {this.state.nombre}
          </label>
      </div>
    );
  }
}
`} size="large" editorProps={{ style: { fontSize: '0.4em', background: '#001628' } }} />

## > ¿Cómo funciona?
Basado en componentes

---

- Clase
```jsx
class Ejemplo extends React.Component { ... }
```
- Función (sin estado ni ciclos de vida por defecto)
```jsx
function Ejemplo(props) {
  return ...;
}
```
- El uso es el mismo
```jsx
<Ejemplo algunParametro='hola' />
```
